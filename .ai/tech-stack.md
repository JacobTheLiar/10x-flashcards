## Backend
- **Java 21**
- **Spring Boot 3.4.4**
- **Spring Security** - do autoryzacji opartej o login i hasło
- **Spring Data JPA** - do obsługi bazy danych
- **Lombok** - redukcja boilerplate kodu
- **MapStruct** - mapowanie obiektów między warstwami
- **Springdoc OpenAPI** - automatyczna dokumentacja API (Swagger)
- **Flyway** - migracje bazy danych

## Frontend
- **Angular 19.2**
- **Angular Material** - komponenty UI
- **RxJS** - obsługa strumieni danych
- **NgRx** (opcjonalnie) - zarządzanie stanem aplikacji
- **Angular JWT** - obsługa tokenów JWT

## Baza danych
- **PostgreSQL 17.4** w kontenerze Docker
- Opcjonalnie: **Redis** - pamięć podręczna, sesje użytkowników

## AI/LLM
- **Ollama** w kontenerze Docker
- Model **Mistral 7B Instruct**
- **LangChain4j** - integracja z LLM w aplikacji Java

## DevOps/Infrastruktura
- **Docker** - konteneryzacja wszystkich komponentów
- **Docker Compose** - orkiestracja kontenerów w środowisku lokalnym
- **GitHub Actions** - CI/CD
- **VPS cal.pl** - hosting

## Testy
- **JUnit 5** - testy jednostkowe
- **Mockito** - mockowanie komponentów
- **Testcontainers** - testy integracyjne z kontenerami
- **Karma/Jasmine** - testy dla Angulara
